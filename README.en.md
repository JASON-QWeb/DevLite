# DevLite

DevLite is a browser extension for frontend debugging and lightweight page editing. It helps you inspect the current page, select elements, adjust styles and copy temporarily, and export the recorded changes as a prompt for an agent to implement in the project.

It is designed for page review, issue reproduction, visual adjustment handoff, and local debugging support.
